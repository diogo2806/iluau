# Suporte

## Onde pedir ajuda

- [GitHub Issues](https://github.com/diogo2806/iluau/issues)

## O que informar

- Seu sistema operacional e a versão do Node.js
- A versão do Roblox Studio
- O cliente de IA que você está usando
- O passo exato que falhou
- Logs ou capturas que ajudem a entender o problema

## Observações

- Este projeto não tem planos pagos nem caminho de upgrade.
- Não publique segredos, chaves de API ou dados privados do projeto em issues públicas.
