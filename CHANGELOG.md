# Changelog

All notable changes to iLuau will be documented in this file.

## [0.4.0] - 2026-06-03

### Added

- Selection tree filter with live search
- Expand all and collapse all controls for the Studio tree
- Property favorites with local persistence
- More readable tree rows with selected-instance highlighting

### Improved

- Property editor UX with better type assistance
- Stronger success and error feedback for property edits and batch updates
- Persisted local property history across Studio sessions

### Updated

- Active docs, support notes, privacy notes, and contributor guidance now match the iLuau project
- The Studio bridge and Codex-facing plugin docs now point to the local `plugins/iluau` workflow
